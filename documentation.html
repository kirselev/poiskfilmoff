Бот по поиску фильмов.

@poiskfilmoff_bot

Данный поможет каждому, кто хочет быстро найти интересующий фильм, постер и ссылку для его просмотра на одной из самых популярных платформ для просмотра фильма. Все просто. Воспользуйтесь простыми командами, введите название фильма и наслаждайтесь просмотром!

Для чего создан

  • Упрощает поиск фильма.
  • Дает возможность выбирать площадку.
  • Выдает лишь самую необходимую информацию.
  • Позволяет

Список команд

  • /start
  • /platform
  • /documentation
  • /help

Описание команд

  1. /start - данная команда позволяет начать диалог с ботом. Эту команду необходимо написать для того, чтобы открыть все возможности бота.
  2. platform - данная команда позволяет производить выбор между площадками Okko и ivi.
  3. /documentation - для просмотра документации напечатайте следующую команду.
  4. /help - в том случае, если что-либо непонятно, просто напечатайте эту команду.

Принцип работы

  • Бот асинхронный. Все сетевые взаимодействия нужно выполняются с помощью aiohttp
  • В качестве библиотеки для работы с API Телеграма, используется aiogram.
  • Для поиска описания, постера и ссылки на фильм, парсятся данные с ресурса для просмотра фильма с помощью библиотеки BeautifulSoup.
  • Понятно, что при вводе названия фильма пользователь может допустить опечатку. Для этого в случае небольших отклонений от названия будет выдаваться наиболее близкие к вводу фильмы.
